# sunmind_thebest
Это приложение по управлению светом, есть включение выключение света, управление яркостью, авто и ручной режим. Также можно добавлять новые световые панели через qr-code и группировать их в зоны. В одной зоне может быть несколько световых панелей, при управлении зоной это касается на все панели которые есть в этой зоне. Также можно управлять светом по времени. И еще есть другие функции.
